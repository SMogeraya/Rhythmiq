# RhythmiQ

RhythmiQ is an Android based E-commerce mobile application which offers music enthusiasts a dynamic and seamless platform to explore, discover and acquire their favourite musical instruments online.
## OBJECTIVES: 
1. **Stores User’s Data in The FireBase:**
- This application stores the required personal data of the user in the Backend 
2. **Delivery point and Payment Choices:**
- In the end the address of the recipient is collected for the delivery of the product. The final section is the payment section for the items purchased by the user. Different payment options will be provided for easy payments for the products
3. **Dynamic and Engaging User Experience**
- Rhythmiq isn't just another shopping app—it's an immersive journey. With its dynamic interface, users are greeted with vibrant visuals and fluid animations that captivate the senses. From browsing products to making purchases, every interaction feels intuitive and engaging, thanks to Flutter's seamless rendering capabilities.
4. **Personalization Redefined**
- Understanding that every shopper is unique, Rhythmiq embraces the power of personalization. Leveraging advanced algorithms, the app curates tailored recommendations based on user preferences, browsing history, and past purchases. Say goodbye to endless scrolling—Rhythmiq puts the perfect products at your fingertips with pinpoint accuracy.
5. **Effortless Navigation and Seamless Checkout**
- With Rhythmiq, navigating through a vast catalog of products is a breeze. Its intuitive search and filter functionalities empower users to find what they need with ease. And when it comes to checkout, Rhythmiq streamlines the process, offering multiple payment options and a secure, hassle-free transaction experience.
6. **Community-Centric Approach**
Beyond its role as a mere marketplace, Rhythmiq fosters a sense of community among its users. Through features like reviews, ratings shoppers can connect, engage, and make informed decisions together, enriching the overall shopping experience.

## Installation

### Cloning the Repository
1. Open your terminal.
2. Navigate to the directory where you want to clone the project.
3. Run the following command:
    ```bash
     https://github.com/YourUsername/YourRepositoryName.git
    git clone https://github.com/SMogeraya/Rhythmiq.git
    ```
4. Navigate into your project’s directory:
    ```bash
    cd Rhythmiq
    ```

### Installing Dependencies
1. Ensure you are in the root directory of the project.
2. Run the following command to install required packages:
    ```bash
    flutter pub get
    ```

## Running the Project
1. Connect your device or start an emulator.
2. Run the following command:
   ```bash
    flutter run
    ```
